# EasyUIFramework

Unity版本 2017.3.1

这是一个简单的Unity UI 框架,功能持续完善中

支持功能:
    1.支持Resources/Assetbundle同步加载UI,先将资源放置在Resources目录或Assetbundles目录下(需运行Build Assetbundle),在BaseUIDefine中注册UI,之后就可以调用BaseUIManager的方法进行加载了
    2.支持计时器,可通过TimeManager方便注册
